# Cube Solid Builder

A browser-based tool for generating and editing 3D cube structures for teaching spatial visualisation (e.g. PSLE-style questions).

Designed for teachers who need **clean, exportable diagrams** without the friction of manual drawing.

---

## ✨ Features

* Interactive **isometric cube builder**
* Click-based editing (no dragging required)
* Generates **random connected solids**
* Export diagrams as **clean PNGs**
* Automatically generates **Top, Front, and Side views**
* Optional **dotted grid** aligned to cube vertices
* Optional **reference wall lines** for spatial clarity
* Ensures **no floating cubes** (physically valid structures)

---

## 🧠 How Editing Works

* **To add cube:**
  Left click a highlighted face to attach one cube to that surface.

* **To remove cube:**
  Right click the visible top face of a stack to remove the top cube.

* Top highlights add a cube on top of a stack; other highlights add a grounded neighbouring cube.

* Floating cubes are not allowed in this builder.

* Optional wall guides can be shown as three simple reference lines anchored to the far back bottom corner of the solid.

---

## 📤 Export Notes

* Dots and wall lines are **not automatically included** in the export.

* Use **Export Settings** to toggle:

  * Dotted grid
  * Wall reference lines

* Two export options:

  * **Download Iso PNG** → Isometric diagram
  * **Download Views PNG** → Top, Front, Side views

---

## 👁 Viewing Modes

* **Hide Guides**
  Removes editing overlays to view the solid clearly.

* **Show Dotted Grid**
  Displays vertex-aligned reference points.

* **Show Walls**
  Displays three reference lines to indicate empty space behind the solid.

---

## ⚠️ Notes for Teachers

* Randomly generated solids may contain **hidden cubes** (not all cubes are visible from the front view).
* This is intentional and useful for testing:

  * spatial reasoning
  * counting hidden cubes
  * interpreting multiple views
