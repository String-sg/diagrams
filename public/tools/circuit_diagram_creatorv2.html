Use Select / Move to select and drag items. Use Wire to draw straight, grid-snapped circuit wires between grid points. Use Arrow to draw a free-position arrow that does not snap to the grid. Use Label Line to draw a free-position straight label line with no arrowhead. Use Grid Arrow to draw an arrow that snaps to the grid. To add text, type into the Text panel, set the font size if needed, click Add Text, then click the canvas to place it. To change an existing textbox, select it, edit the text or font size in the Text panel, and click Update Selected. Use Rotate 90° or Rotate 180° to rotate the selected component; if nothing is selected, the chosen rotation is used for the next component placed. Use Battery cells in series to change the number of cells before placing a battery. Use the +, −, and 100% buttons above the canvas to zoom in, zoom out, or reset zoom; use Centre View to recenter the current drawing.

Tips: Delete removes the selected item. R rotates 90°. Ctrl/Cmd+Z undo. Ctrl/Cmd+Shift+Z redo. Wires snap to the grid; arrows and label lines do not.
