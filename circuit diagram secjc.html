Use Select / Move to drag components. Use Wire to draw straight snapped segments between grid points. Use Arrow to draw free-position arrows that do not snap to the grid. Use Grid Arrow for a grid-snapped arrow. Crossings count as connections. Copy and paste commands work only on arrows. Delete removes the selected item. Ctrl+C copy selected arrow. Ctrl+V paste arrow. Ctrl+Z undo. Ctrl+Shift+Z redo.

Tips: Delete removes the selected item. Ctrl/Cmd+C copy selected arrow. Ctrl/Cmd+V paste arrow. Ctrl/Cmd+Z undo. Ctrl/Cmd+Shift+Z redo.
