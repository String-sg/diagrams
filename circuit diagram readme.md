# Circuit Diagram Creator Suite

A browser-based set of HTML tools for making **classroom-friendly circuit diagrams** quickly.

This suite currently includes:

* **Symbol Circuit Diagram Creator** for standard science symbols
* **Object Circuit Diagram Creator** for apparatus-style circuit drawings

## What it is for

Use these tools to make clean circuit diagrams for:

* worksheets
* slides
* notes
* simple assessment visuals

The diagrams can be exported as PNG images for easy use in PowerPoint or documents.

## How to use

1. Open the suite HTML file in a web browser.
2. Choose a tool from the side panel.
3. Click to place components or draw wires.
4. Use select / move / rotate as needed.
5. Export the finished diagram as PNG.

## What is included

### Symbol creator

Use this when you want conventional circuit symbols such as:

* wires
* bulbs
* batteries
* switches

### Object creator

Use this when you want circuit parts drawn more like real objects rather than symbols.

## Important note

This suite is for **circuit diagrams only**.

It does **not** currently include:

* electromagnets
* full electrical simulation
* automatic checking of circuit correctness

## Why it exists

The aim is to give teachers a lightweight way to build neat circuit visuals without needing heavy drawing software.
